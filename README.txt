# muzon
Программа для воспроизведения большой музыкальной коллекции различными способами, подобна mp3-master ,только имеет GUI.

Программа для управления воспроизведением музыки ,хранящейся
в различных каталогах.Она использует готовые консольные проигрыватели типа mpg123. Проверена в системе WindowsXP.
Написана на языке Racket5.3
   Вся информация необходимая для работы программы находится
в файле upr.conf . На первой строчке находится путь к используемому
проигрывателю. Далее расположены файлы со списками музыкальных
директорий : "c:/Free/mpg123/bin/mpg123.exe "
             "all.txt"
             "via.txt"
             "bardy.txt"
             ..........
             
   Файлов со списком музыкальных директорий может быть сколько угодно. Они представляют из себя списки путей к нужным папкам:
   "d:/Musik/Mozart/" "d:/Musik/Boka/" "d:/Musik/Vivaldi/"....
 Если папок много ,то составление списка может оказаться очень трудоёмким. Для облегчения этого процесса написаны две небольшие программки:index.rkt и index2.rkt. Они запускаются в консоли ,а в качестве
    аргумента передаётся путь к корневой папке : 
                 index.exe "g:/Musik"
index составляет списки для директорий где музыкальные файлы хранятся на
втором уровне: общая папка->папки альбомов->музыкальные файлы. Index2 
применяется там где музыкальные файлы находятся на третьем уровне:
 общая папка->общие папки->папки альбомов->музыка. Из полученных таким
 образом файлов списков необходимо удалить все скобки и только после этого они могут применяться.
      Существует три режима воспроизведения : "регулярный" - все папки 
проигрываются в том порядке в каком они записаны;
    "случайный" - папка выбирается случайно ,файлы в папке тоже выбираются
    случайно;
    "случйно-одиночный" - папка выбирается случайно из неё проигрываются два случайно выбранных файла и далее следует переход к другой папке...
    
   Для начала необходимо получить исполнительные файлы : player.exe ,play.exe ,play-rand.exe ,play-uno.exe. Для этого в консоли набираем :
    
          raco exe <path:>/player.rkt и так далее...
          
   Потом составляем хотя бы один текстовый файл со списком директорий и заполняем файл upr.conf .
   После этого можно запускать player.exe ! Появится простенькое
   GUI, программа просмотрит рабочую директорию и занесёт все
   найденные списки с музыкой в лист бокс слева , выбирете нужный
   список ,потом режим воспроизведения в центре и нажмите на кнопку "старт" справа. Самая крайняя кнопка справа - "play dir"
   служит просто для воспроизведения файлов находящихся в какой-то
   конкретной директории.
   После этого появляется консоль с выбранным плеером и начинается
   воспроизведение по выбранной программе. Всё дальнейшее управление зависит от консольного плеера.Выключить всегда можно
   набрав сочетание Ctrl/c .
